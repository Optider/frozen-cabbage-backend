# Frozen Cabbage
A Next.js service that provides the generated boilerplate code.</p>

Currently supported templates:
- Frontend
  - Vanila JS
  - Vue.js
  - Capacitor (Android + iOS)
- Backend
  - Express.js
  - Flask

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
